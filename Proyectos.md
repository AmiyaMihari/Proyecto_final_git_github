## 📌 **Proyecto 1: Optimización de rutas con Python y Machine Learning usando Clusters**
- **Descripción:**  
  Utilicé Python junto con Scikit Learn para optimizar rutas, utilizando algoritmos de machine learning para agrupar en clusters (k-means) para crear zonas automáticas de repartición, basados en criterios de prioridad y proximidad, utilizando APIs de Google Maps y el cálculo de la Matriz de Distancia.
- **Tecnologías Utilizadas:**  
  ![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
  ![Google Cloud](https://img.shields.io/badge/Google_Cloud-4285F4?style=for-the-badge&logo=google-cloud&logoColor=white)  
- **Características Destacadas:**  
  - ✨ Optimización dinámica semanal
  - ✨ Visualización de las rutas en el mapa
